## Findings — FYP Management Design Sprint

### Project Context
- **App**: Java Swing MVC, 3 roles, 3 core UCs, ~25 Swing frames
- **Screens**: Login, StudentDashboard, SupervisorDashboard, AdminDashboard, PeerReview, SubmitProposal, Milestones, etc.
- **Aesthetic Direction**: Institutional Modernism — dark, editorial, data-dense, Swiss-grid
- **Fonts chosen**: DM Serif Display (headings) + DM Sans (body) + JetBrains Mono (data)
- **Accent**: Vermillion #D94F2E (action) + Steel-blue #3B8FCC (info)

### Poster
- Already produced: fyp_management_poster.png (2480x3508)
- Philosophy doc: design_philosophy.md

### Brand Voice (Sentry/Plain-Speech hybrid for academic tool)
- Plain Speech for all UI labels
- No exclamation marks; no "please"; no "success!"
- Tone: Trusted institutional tool, not startup cheerfulness

### Key Design Decisions
| Decision | Choice | Rationale |
|----------|--------|-----------|
| Color mode | Dark only | University lab environments; reduces eye strain |
| Grid | 4px base, 12-col | Swiss precision, aligns with institutional feel |
| Sidebar | Monolithic slab | Ties to poster philosophy; role-color indicator |
| Typography scale | 12/14/16/20/28/40 | DM Sans at these sizes is extremely legible |
